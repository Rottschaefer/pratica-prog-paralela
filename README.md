# Parallel Programming Laboratory Repository

## Overview

This repository contains code examples and exercises developed for the study of **Parallel Programming** in the course **Parallel Programming Laboratory** at **Universidade Federal Fluminense (UFF)**, Bachelor's degree in Computer Science.

## Purpose

The code in this repository is designed to demonstrate fundamental concepts of parallel programming, including:

- Message passing with MPI
- Thread parallelism with OpenMP
- Process communication and synchronization
- Algorithm parallelization strategies
